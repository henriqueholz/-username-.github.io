Criando um blog com Hexo e Github

Segue um tutorial simples e rápido de como criar um blog e hospedar no GitHub de graça:

Quick Start

Instale o Hexo

Crie um projeto para sua página no GitHub

Entre na pasta criada e instale os arquivos que faltam

Configure seu código para usar o Git

Abra seu projeto, abra o arquivo _config.yml e faça a seguinte edição com o endereço do seu blog:

deploy:
type: git
repo: git@github.com:henriqueholz/henriqueholz.github.io.git
branch: master
